### Shipping now

**[frugon](https://github.com/Rodiun/frugon)** — free, open-source LLM cost analyzer. Point it at your call logs and see, on your machine, how much you'd save by switching or routing models. Your data never leaves your machine.

```bash
uvx frugon analyze --demo        # try it in 5 seconds
pipx install frugon              # then point it at your own logs
```
